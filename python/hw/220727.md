### Instance Method
.sort() 와 같이 문자열, 리스트, 딕셔너리 등을 조작 할 때 사용하였던 것들은 클래스에 정의된 메서드들이었다. 이처럼 문자열, 리스트, 딕셔너리 등을 조작하는 메서드를 최소 3가지 이상 그 역할과 함께 작성하시오.

```python
.upper() # 대문자로 변환.
.lower() # 소문자로 변환.
.title() # 단어의 첫 문자를 대문자로 변환.
```
