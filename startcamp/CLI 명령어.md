# CLI INTRO

## 각종 명령어들

### 파일 생성

**touch**

``` bash
$ touch {파일명}
```

---

### 파일 목록 확인

**ls**

``` bash
$ ls
$ ls -l 파일 상세 확인
$ ls -a 상위파일도 나옴
```

### 터미널 클리어

``` bash
 ctrl + l
```



- `pwd` : 현재의 위치를 확인한다.
- `cd` : 폴더에 진입한다.
- `open .` : 현재 폴더를 파일 탐색기로 연다.
- `mkdir` : 새로운 폴더를 생성한다.
- `touch` : 새로운 파일을 생성한다.
- `cat` : 파일의 내용을 출력한다.
- `rm` : 폴더나 파일을 삭제한다.
- `mv` : 폴더나 파일의 위치를 이동하거나 이름을 변경한다.
- `cp` : 폴더의 파일을 복사한다.
- `sudo` : 관리자 권한으로 다음 명령어를 실행시킨다.
